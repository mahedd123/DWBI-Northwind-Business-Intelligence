# DWBI-Northwind-Business-Intelligence
Data Warehousing and Business Intelligence project using SQL Server, SSAS, Excel OLAP, and Power BI.

## Overview

This project demonstrates a complete Business Intelligence solution using the Northwind dataset.

## Architecture

Northwind Dataset
↓
ETL Process
↓
Data Warehouse
↓
SSAS Cube
↓
Excel OLAP Analysis
↓
Power BI Dashboard

## Technologies

- SQL Server
- SSIS
- SSAS
- Power BI
- Excel

## Features

- Data Warehouse Design
- ETL Development
- OLAP Cube
- Interactive Dashboards
